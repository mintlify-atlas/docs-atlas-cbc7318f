## What is Nextly?

Nextly is an open-source, AI-powered tool that turns a natural language description into a complete, runnable **Next.js 15** application — with TypeScript, Tailwind CSS, and Shadcn UI — in one shot. It's the fastest way to prototype web apps without writing boilerplate.

<Info>
  Nextly is currently in **beta**. Generation can take up to 10 minutes on free-tier models. Preview URLs expire after 10–30 minutes — download your code promptly.
</Info>

## What you can build

Nextly excels at small-to-medium scoped projects. Here are common use cases:

- **Landing pages** — Hero, features, pricing, testimonials, footer sections
- **Portfolios** — Personal sites with project grids and contact forms
- **Dashboards** — Admin panels with charts, data tables, and sidebars
- **Blogs** — Article listings, individual post pages, author info
- **Auth pages** — Login, sign-up, forgot password flows with validation
- **E-commerce pages** — Product pages, cart UIs, category listings
- **Component demos** — Isolated UI components with state and interaction

<Tip>
  Keep projects focused and well-scoped. Nextly works best when your prompt describes a single, coherent page or feature — not a multi-page enterprise application.
</Tip>

## Tech stack overview

Nextly itself is built with the same stack it generates apps in:

| Layer | Technology |
|---|---|
| Framework | Next.js 15 with App Router |
| Language | TypeScript |
| Styling | Tailwind CSS + Shadcn UI |
| Auth | Clerk |
| Database | PostgreSQL + Prisma |
| API | tRPC |
| Background jobs | Inngest |
| Code execution | E2B sandboxes |
| AI models | Grok 4, GPT Codex, Gemini 2.5 Flash |
